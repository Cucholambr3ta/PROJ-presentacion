# CLAUDE.md — PROJ-presentacion

**Identidad del Proyecto**: Sitio de Presentación de Título Elite (OLYMP-IA).
**Tipo**: Web Application (React/Vite).
**Misión**: Exponer el trabajo de grado con un nivel estético deslumbrante y profesional.

## GUÍA DE DESARROLLO
- **Code Style**: JSDoc mandatory. Arrow functions preferred for functional components.
- **Naming**: camelCase for files/variables. PascalCase for Components.
- **Directory Structure**: Separate concerns between components and hooks.
- **Styling**: Use `Design Tokens` and CSS variables in `src/styles`.
- **Animations**: Framer Motion for page/scroll transitions.

## COMANDOS LCM
- `npm run dev`: Iniciar el entorno de desarrollo local.
- `npm run build`: Generar el bundle de producción para Vercel.
- `npm run lint`: Validar la calidad del código.
- `npm run preview`: Vista previa del build local.
